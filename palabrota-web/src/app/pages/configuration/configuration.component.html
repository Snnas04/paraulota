Dificultad

Selecciona la dificultad deseada.

Varia la cantidad de letras que puede tener la palabra y la cantidad de intentos para adivinar dicha palabra.

Color

Modifica le color principal de la web

Objetivo

Se tiene que insertar una palabra que rellene todos los huecos.
Una vez enviada se colorea, cada color tiene un significado.

m

Verde: Está en el lugar correcto

s

Naranja: Está presente en la palabra objectivo, pero mal posicionada

v

Gris: No se encuentra en la palabra objetivo

Modos de Juego

Normal

Se genera una palabra
respecto a la dificultad
seleccionada y se tiene
que adivinar en un número
determinado de intentos.

Diario

Cada dia hay una palabra
y se tiene que adivinar,
no afecta la dificultad
que tengamos seleccionada.
Simpre hay seis intentos.

Buscar

Se puede buscar la palabra que quieras y ver todas las definicions, sinonimos, antonimos, etc.
